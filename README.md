# WebCraft Development

Website Profesional yang Membantu Bisnis Anda Tumbuh

Bangun kepercayaan pelanggan melalui website yang modern, cepat, dan dirancang sesuai kebutuhan bisnis Anda. Mulai dari UMKM, company profile, landing page, hingga toko online, WebCraft Development siap membantu menghadirkan solusi digital yang profesional.

Perubahan terbaru di repo:
- Menambahkan situs statis (index.html) dengan desain yang diperbarui.
- Menambahkan logo (assets/logo.svg) dan favicon (assets/favicon.svg).
- Menambahkan screenshot portfolio placeholder: assets/portfolio/indoprint-screenshot.svg dan sepulang-screenshot.svg.
- Menambahkan form kontak statis yang mengirim email via FormSubmit.co (action: https://formsubmit.co/info@webcraftdevelopment.com). Ganti action jika Anda menggunakan layanan lain.
- Menambahkan metadata SEO / Open Graph di index.html.

Jika Anda ingin mengganti screenshot portfolio dengan gambar nyata, unggah file gambar ke folder assets/portfolio dan beri nama yang sama (indoprint-screenshot.* dan sepulang-screenshot.*), atau perbarui path di index.html.

Cara menampilkan di GitHub Pages:
1. Buka Settings → Pages pada repository.
2. Pilih branch: main (folder: / (root)).
3. Simpan. Setelah beberapa menit, situs akan aktif di https://<username>.github.io/<repo>/ atau domain kustom jika dikonfigurasi.

Selanjutnya yang bisa saya bantu (pilih yang Anda mau):
- Ganti placeholder screenshot dengan file PNG/JPG asli (unggah gambar) dan optimasi ukuran.
- Sesuaikan warna, font, atau tata letak lebih lanjut.
- Buat template Next.js/React untuk kemampuan dinamis dan admin dashboard.
- Bantu menghubungkan form ke layanan lain (Formspree, Getform, Netlify Forms, Typeform) dan tes pengiriman.
- Siapkan CNAME dan panduan domain kustom / HTTPS.
